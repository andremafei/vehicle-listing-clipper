# CRM lead panel (in-bundle)

CRM verify/create UI lives under `src-crm-filler/` and is **bundled into** the main Vehicle Listing Clipper userscript. There is no separate Tampermonkey install.

On `https://crm.flexicar.pt/*` (and local LeadDesk at `http://127.0.0.1:4173/crm/*`) the same script mounts the CRM panel instead of the listing clipper.

Install: see the root [README](../README.md) (`vehicle-listing-clipper-local.user.js` / production `.user.js`).

API notes: [docs/crm-api-from-hars.md](../docs/crm-api-from-hars.md).
